## 📌 Descripción del Proyecto
Este proyecto implementa un pipeline de Ingeniería de Datos de extremo a extremo (End-to-End) enfocado en la manipulación automatizada y análisis de grandes volúmenes de datos. Se procesan múltiples conjuntos de datos utilizando la librería **Pandas** en Python para ejecutar ciclos completos de extracción, transformación profunda y carga de datos limpios.

---

## 🛠️ Arquitectura y Procesamiento (ETL)

*   **Extracción:** Carga automatizada de datasets desde archivos planos (`.csv`), gestionando de manera eficiente la estructura inicial.
*   **Transformación (Data Wrangling):**
    *   Normalización y limpieza de tipos de datos, strings y registros nulos.
    *   Tratamiento estadístico avanzado para la detección de anomalías e insights clave.
*   **Carga:** Consolidación de datos procesados, garantizando un almacenamiento óptimo para etapas posteriores de visualización.

---

## 💻 Tecnologías Utilizadas

*   **Python 3.13** - Lenguaje principal para la lógica de automatización.
*   **Pandas** - Manipulación estructural de datos y optimización de memoria.
*   **Jupyter Notebooks** - Entorno de desarrollo para prototipado rápido y análisis exploratorio (EDA).
*   **Visual Studio Code** - IDE de desarrollo y control de versiones con Git.
